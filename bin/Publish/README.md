# HGmusic - 游戏语音音效板

一款轻量级的音效板工具，可以将音频文件播放到游戏语音频道中，让你的队友听到音乐/音效。

## 功能特性

- 🎵 支持 MP3、WAV、AIFF、AAC、M4A 格式的音频文件
- 🎮 通过 VB-Cable 虚拟声卡将音频路由到游戏语音频道
- ⌨️ 为每个音效卡片设置独立快捷键，游戏中一键触发
- 🎤 PTT 自动按键：播放音效时自动模拟按住语音键
- 🎤 麦克风混音：使用虚拟声卡时仍能正常说话
- ✂️ 波形裁剪：设置音频的播放起止区间
- 🖥️ 系统托盘运行：最小化后不干扰游戏体验
- 📋 预设管理：保存和切换多套配置

## 使用方法

### 下载运行

下载 `HGmusic.exe`，双击即可运行（需要管理员权限）。

### 基本设置

1. 安装 VB-Cable 虚拟声卡（程序内可一键安装）
2. 在游戏设置中将麦克风设备设为 **CABLE Output**
3. 在程序的输出设备中选择 **CABLE Input**（游戏输出）和你的耳机/音箱（本地监听）
4. 添加音效文件，设置快捷键，即可使用

## 系统要求

- Windows 10 / 11
- 需要管理员权限运行（用于全局热键和模拟按键）
