# 🎵 HGmusic — 游戏语音音效播放面板

> 在游戏中一键播放音效，让队友都能听到！

HGmusic 是一款专为游戏玩家设计的 Windows 桌面音效面板。通过虚拟音频线缆（VB-Cable）将音效路由到游戏语音频道，队友可以实时听到你播放的音乐、音效和语音片段。同时支持麦克风混音，确保你的正常语音通话不受影响。

---

## ✨ 功能特性

### 🎮 音效播放
- 支持 **MP3、WAV、AIFF、AAC、M4A** 等主流音频格式
- 双设备同时输出：游戏虚拟麦克风 + 本地监听（耳机/音箱）
- 可配置播放时长（1-60 秒），到时自动停止
- 同一时间只播放一个音效，新的播放自动停止上一个

### ⌨️ 全局热键
- 基于底层键盘钩子的全局热键，不干扰其他应用的快捷键
- 每个音效卡片独立绑定热键，支持 Ctrl/Shift/Alt 组合键
- 双重输入检测（低级钩子 + 轮询），确保按键不丢失
- 热键冲突检测，避免与全局功能键重复

### 🎙️ PTT 自动化
- 播放音效时自动模拟按下 PTT 键，队友即时听到声音
- 支持 PTT 常开模式（手动长按语音键）
- 基于音频检测的语音活动识别（RMS 阈值检测）

### 🔊 麦克风混音
- 自动捕获物理麦克风并混入虚拟音频设备
- 播放音效的同时队友仍能听到你的语音
- 智能排除 VB-Cable 设备，自动选择正确的物理麦克风

### 🎨 波形编辑器
- 可视化波形显示，直观选取音频片段
- 支持设置起止点进行音频裁剪
- 精确控制音效的播放区间

### 📦 VB-Cable 管理
- 自动检测 VB-Audio Virtual Cable 安装状态
- 一键下载安装，无需手动操作
- 详细的安装进度引导

### 🎯 预设系统
- 支持多套配置预设，一键切换
- 每个预设独立存储设备选择、音量、热键绑定等所有设置
- 支持创建、复制、重命名、删除预设

### 🌙 暗色主题 UI
- 全自绘暗色主题界面，风格统一
- 音效卡片支持悬停动画、播放脉冲指示、渐变背景
- 滑出式设置侧面板，带动画过渡效果
- 系统托盘集成，最小化到托盘不占任务栏
- 首次使用引导向导，快速上手

## 🚀 快速开始

### 方式一：直接运行（推荐）

1. 下载发布包中的 `HGmusic.exe`
2. 右键以 **管理员身份运行**（程序会自动请求提权）
3. 首次启动会进入 **设置向导**：
   - 导入音效文件
   - 设置播放热键
   - 配置虚拟音频设备
4. 完成向导后即可使用！

### 方式二：从源码构建

```bash
# 克隆仓库
git clone <仓库地址>
cd music_games

# 构建项目
dotnet build HGmusic.csproj

# 运行
dotnet run --project HGmusic.csproj
```

## 📖 使用指南

### 添加音效
- **拖拽导入**：直接将音频文件拖放到主窗口
- **点击添加**：点击主界面的 "+" 卡片，选择音频文件

### 绑定热键
1. 右键点击音效卡片
2. 选择「设置热键」
3. 按下想要绑定的按键组合（如 `Ctrl+1`）

### 裁剪音频
1. 右键点击音效卡片
2. 选择「设置裁剪」
3. 在波形编辑器中拖动选取播放区间
4. 确认保存

### 配置设备
1. 点击界面右上角的齿轮图标打开设置面板
2. **游戏输出设备**：选择 `CABLE Input`（VB-Cable 虚拟设备）
3. **本地监听设备**：选择你的耳机或音箱
4. **物理麦克风**：程序会自动排除虚拟设备并选中你的麦克风

### 游戏内设置
在游戏的音频设置中，将 **麦克风/语音输入** 设为 `CABLE Output`，即可接收 HGmusic 播放的音效。

### 快捷操作
| 操作 | 默认快捷键 |
|------|-----------|
| 触发音效播放 | 各卡片独立热键 |
| 停止当前播放 | `Ctrl + S` |
| PTT 常开 | `V`（长按） |

## ❓ 常见问题

**Q: 为什么需要管理员权限？**
A: 全局热键钩子和 PTT 键盘模拟（SendInput）需要管理员权限才能在其他以管理员运行的程序（如游戏）中正常工作。

**Q: 不安装 VB-Cable 能用吗？**
A: 可以使用音效播放和本地监听功能，但无法将音效路由到游戏语音频道。VB-Cable 是实现虚拟麦克风的关键组件。

**Q: 队友听不到我的音效怎么办？**
A: 请检查：
1. VB-Cable 是否已安装（程序内可一键安装）
2. 游戏内麦克风是否设置为 `CABLE Output`
3. HGmusic 设置中游戏输出设备是否选择 `CABLE Input`
4. PTT 功能是否已启用

**Q: 支持哪些音频格式？**
A: MP3、WAV、AIFF、AAC、M4A（由 NAudio 库支持）。
